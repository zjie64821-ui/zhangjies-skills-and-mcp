# 文件整理助手

智能整理文件和文件夹，支持查找重复、建议结构、自动化清理。

## 工作流

### 1. 了解范围
确认目标目录、主要问题、需要避免的文件、整理力度

### 2. 分析现状
```bash
ls -la [目录]
du -sh [目录]/* | sort -rh | head -20
find [目录] -type f | sed 's/.*\.//' | sort | uniq -c | sort -rn
```

### 3. 查找重复
```bash
find [目录] -type f -exec md5 {} \; | sort | uniq -d  # 精确重复
find [目录] -type f -printf '%f\n' | sort | uniq -d     # 同名文件
```

### 4. 提出方案
展示当前状态 → 提议结构 → 变更列表 → 等待确认

### 5. 执行整理
创建文件夹 → 移动文件 → 重命名 → 删除重复（需确认）

## 整理维度
- **按类型**: 文档/图片/视频/压缩包/代码/表格/演示
- **按用途**: 工作/个人、活跃/归档、项目特定
- **按时间**: 当前/往年/归档

## 命名规范
- 文件夹：清晰描述性名称，避免空格
- 文件：`YYYY-MM-DD - 描述.ext` 格式
- 清理下载残留：`document-final-v2 (1).pdf` → `document.pdf`

## 安全规则
- 删除前必须确认
- 记录所有移动操作以便撤销
- 保留原始修改日期
- 遇到意外情况暂停询问
