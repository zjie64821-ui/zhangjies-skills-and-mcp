# Obsidian 笔记库控制

通过 Obsidian Local REST API 插件直接控制 Obsidian 笔记库。

## 功能
- **搜索笔记**: 按关键词搜索
- **创建/更新笔记**: 新建或覆盖已有笔记
- **追加内容**: 在笔记末尾添加内容（不覆盖）
- **读取笔记**: 获取指定笔记内容
- **当前笔记操作**: 读取或更新当前打开的笔记
- **命令面板**: 触发 Obsidian 命令（如切换深色模式）

## 使用示例
- "在 Obsidian 中搜索 '项目计划'"
- "在我的每日笔记中添加任务：'完成报告'"
- "读取 Obsidian 中 '研究笔记.md' 的内容"
- "在当前打开的笔记中追加对话摘要"

## 前置条件
需要安装 Obsidian Local REST API 插件并启用。
